Fullstack development

I have been exposed to fullstack web development for the passed 4 years and the journey has been enriching. In all fullstack tools out there, I find Python-django to be the simplest and friendliest to use. It makes it easy to develop scalable and integrated systems because of python's ability to pip install third-party systems. For instance, if a client wants a system that is used to locate GSM device, you simply run "pip install geopy" to access geagraphycal co-ordinates of the device, making python one of the most powerful languages when coming to real-life solutions. The backend data structure is easy to develop, allowing easy CRUD processes and dynamic data capturing and sorting. So, I would like to continue working on projects that employ the use of Python-django to grow more experience.

The fullstack project I worked on is a Customer Dashboard system that users use to place, process and delete orders. Models with different tags where used for searching and sorting. I used bootstrap for most of frontend design.
